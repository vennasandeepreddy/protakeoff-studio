# ProTakeoff Studio

## How to Run Locally

1. **Install Node.js**: Ensure you have Node.js installed (version 16 or higher recommended).
2. **Install Dependencies**: Open your terminal in this folder and run:
   ```bash
   npm install
   ```
3. **Start Development Server**: Run the following command to start the app:
   ```bash
   npm run dev
   ```
4. **Open Browser**: The app should automatically open at `http://localhost:3000`.

## Build for Production

To create a production build:
```bash
npm run build
```
